## 🔓 PREMIUM FONTS (Need Adobe Fonts / Purchase)

### **Adobe Caslon Pro** (5 weights needed)
- **Access:** Included with Adobe Creative Cloud subscription
- **Download:** Open Adobe Creative Cloud → Fonts → Search "Adobe Caslon Pro" → Activate all weights
- **Or:** https://fonts.adobe.com/fonts/adobe-caslon

### **Acumin Pro** (3 weights needed)
- **Access:** Included with Adobe Creative Cloud subscription
- **Download:** Adobe Creative Cloud → Fonts → Search "Acumin Pro"
- **Or:** https://fonts.adobe.com/fonts/acumin

### **Proxima Nova Extra Condensed** (4 weights needed)
- **Access:** Premium font (not free)
- **Download:** https://fonts.adobe.com/fonts/proxima-nova (if you have Adobe subscription)
- **Or Purchase:** https://www.myfonts.com/collections/proxima-nova-font-mark-simonson-studio

### **Franklin Gothic** (9 variations!)
- **Note:** You have "FranklinGothic.rar" in C:/Windows/Fonts/ - need to extract it
- **ATF Version:** Professional font, may need purchase
- **LT Pro Version:** https://www.linotype.com/fonts/franklin-gothic-family
- **URW Version:** May be included with some Adobe packages

### **Aileron** (Free - alternate source)
- **Download:** https://www.behance.net/gallery/20721059/Aileron-Free-Typeface
- **Or:** https://github.com/dotcolon/aileron/releases

### **Poor Richard**
- **Access:** Classic Windows font (should already be on your system)
- **If missing:** Search Windows fonts or download from font archives

## 🎯 QUICK START INSTRUCTIONS

### 1. Extract Franklin Gothic
```bash
# In C:/Windows/Fonts/
Right-click FranklinGothic.rar → Extract Here
```

### 2. Activate Adobe Fonts
- Open Adobe Creative Cloud app
- Go to Fonts tab
- Search and activate:
  - Adobe Caslon Pro (all weights)
  - Acumin Pro (all weights)
  - Proxima Nova Extra Condensed (all weights)

### 3. Upload to Canva
- Canva → Brand Kit → Upload Fonts
- Upload all .ttf and .otf files from this folder
- Upload activated Adobe fonts from: C:/Users/[YourName]/AppData/Local/Adobe/CoreSync/plugins/livetype/r/

## 📂 Where Adobe Fonts Install
After activating in Adobe Creative Cloud, fonts go to:
```
C:/Users/Ben/AppData/Local/Adobe/CoreSync/plugins/livetype/r/
```

Look for:
- ACaslonPro-*.otf
- AcuminPro-*.otf  
- ProximaNova*.otf

Copy these to this fonts_for_canva folder for easy upload to Canva!
